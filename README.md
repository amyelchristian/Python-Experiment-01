# Python Experiments - Experiment 01

A collection of beginner Python programs covering core concepts like user input, string manipulation, arithmetic, and working with dates.

## Problems

| File | Description |
|------|-------------|
| [problem01.py](problem01.py) | Repeat a string `n` times based on user input |
| [problem02.py](problem02.py) | Calculate the average of 3 numbers |
| [problem03.py](problem03.py) | Find the year the user will turn 100 years old |
| [problem04.py](problem04.py) | Calculate simple interest and total amount payable |
| [problem05.py](problem05.py) | Reverse a number entered by the user |

## Running a Program

```bash
python problem01.py
```

## Requirements

- Python 3.x
